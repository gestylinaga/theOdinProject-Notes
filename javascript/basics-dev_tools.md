### Opening Dev Tools
Three ways:
1. From the `Chrome Menu` > `More Tools` > `Developer Tools`
2. Right-click anywhere on the page and select `Inspect`
3. Keyboard shortcuts: `F12` or `CTRL + Shift + C`
